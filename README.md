# 엑셀 변경사항 확인 웹앱

행 위치 변경을 무시하고, **값 변경**, **배경색 변경**, **추가된 행**을 잡아내는 Streamlit 웹앱입니다.

## 사용 시나리오

1. **기준 엑셀 업로드** → 스냅샷(baseline.json) 파일 다운로드/보관
2. **수정된 엑셀과 스냅샷** 업로드 → 변경사항/추가된 행 확인 및 **엑셀 보고서** 다운로드

## 주요 기능

- ✅ **행 재정렬 감지**: 순서가 달라도 정확히 매칭
- 🎨 **색상 변경 감지**: 배경색(채우기) 변경 추적 (30+ 색상 지원)
- 📊 **전체 열 비교**: 모든 열의 값과 색상 변경 감지
- 🔍 **결과 필터링**: 변경 내용 검색 및 카테고리별 표시/숨김
- 💾 **다운로드 옵션**:
  - 전체 결과 다운로드
  - 변경/추가/제거된 항목만 다운로드
- 📁 **로컬 폴더 지원**: 파일 업로드 또는 로컬 폴더에서 직접 선택
- ⏳ **진행 상황 표시**: 대용량 파일 처리 시 실시간 진행률 표시

## 설치 방법 (로컬)

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 배포 (Streamlit Community Cloud)

1. 저장소의 이 프로젝트 파일들을 업로드 (app.py, requirements.txt 등).
2. [Streamlit Community Cloud](https://streamlit.io/cloud) 접속 → New app → 저장소/브랜치/파일(app.py) 선택 → Deploy.
3. 앱이 기동되면, 링크에서 기존 스냅샷을 먼저 선택한 다음, 새 파일을 선택해 비교하세요.

## 기능 메모

- **정규화 옵션**: 앞뒤 공백 무시, 대소문자 구분 설정 가능
- **다중 시트 지원**: 기준/비교 파일에서 원하는 시트 선택 가능
- **컬러 매핑**: Yellow, Red, Green, Blue, Orange, Purple, Gray 등 30+ 색상 친화적 이름 표시
- **검색 기능**: 변경 내역에서 키워드 검색
- **에러 처리**: 빈 파일, 잘못된 경로 등 다양한 에러 상황 대응

- `README.md`: 이 파일
